Download

The source repository,including packaged windows binaries is available from http://hci.iwr.uni-heidelberg.de/MIP/Software/

Installation

Using the binary, just place it in a convenient location, take care that the Models folder is in the same place.

Building from Source

The following illustrates how to build the examples, tests and distribution packages. Bundling libfbi requires a working CMake build system (available from http://cmake.org/) and CMake >= 2.6.

With cmake in the system path, the build process is 

 Either
 tar xvzf biceps-xxxxxxx.tar.gz
 or
 git clone git://github.com/buote/BICEPS.git

 mkdir biceps
 cd biceps
 ccmake ../biceps
 make && sudo make install

 Example line:
 biceps --mgf pickedpeaks_no_pp_Helatest_with_none_resultRawList.mgf --fasta ipi.CHICK.v3.54.fasta 
 For other parameters, just execute biceps.
